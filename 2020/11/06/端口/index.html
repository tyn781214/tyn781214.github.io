端口

发表于 2020-11-06

基本知识

PC机中，和CPU通过总线相连的芯片除各类存储器外还有：

​ 1.各类接口卡上的接口芯片：控制接口卡进行工作

​ 2.主板上的接口芯片：对部分外设进行访问

​ 3.其他芯片：存储相关的系统信息或进行相关的输入输出处理

上述芯片都有一组可被CPU读写的寄存器：

​ 1.都和CPU总线相连

​ 2.CPU对其进行读写通过控制线向它们所在芯片发出读写命令

CPU将这些寄存器当作端口，对它们统一编址。

所以CPU可以直接读写的数据来源有：1.CPU内部寄存器、2.内存单元、3.端口

端口的读写

PC机中，CPU最多可定位64K个不同的端口，端口地址的范围为0～65535。

端口的读写指令：IN和OUT指令

IN指令

输入指令: IN （I/O—> CPU）

长格式： IN AL, PORT （字节） 

IN  AX, PORT      （字）

执行操作： (AL) <— ( PORT ) （字节）

​ (AX) <— ( PORT+1, PORT )

短格式： IN AL, DX （字节） (AL端口范围：0~255)

IN  AX, DX      （字）    (AX端口范围：256~66535)

执行操作： (AL) <— ( (DX) ) （字节） 

(AX)   <—( (DX)+1, (DX) )

OUT指令

输出指令: OUT （I/O<—CPU）

长格式： OUT AL, PORT （字节） 

OUT  AX, PORT      （字）

执行操作： (PORT) <— ( AL) （字节）

​ (PORT+1, PORT ) <— (AX )

短格式： OUT DX, AL （字节） (AL端口范围：0~255)

OUT  DX, AX      （字）    (AX端口范围：256~66535)

执行操作： ((DX))) <— ( AL ) （字节） 

((DX)+1, (DX) )   <—( AX )

注意事项

IN/OUT指令中，只能使用ax 或者al来存放从端口中读入的数据或者要发送到端口中的数据，访问 8 位端口用al, 访问 16 位端口用 ax

CPU访问内存和访问端口的区别

访问内存：mov ax,ds:[8]

相关操作：

​ 1.cpu通过地址线将地址信息8发出；

​ 2.通过控制线发出内存读命令，选中存储器芯片，并通知它，将从中读取数据；

​ 3.存储器将8号单元中的数据通过数据线送入cpu

例如：访问端口：in al,60h

相关操作：

​ 1.cpu通过地址线将地址信息60h发出；

​ 2.通过控制线发出端口读命令，选中端口所在芯片，并通知它，将从中读取数据；

​ 3.端口所在芯片将60h端口中的数据通过数据线送入cpu
