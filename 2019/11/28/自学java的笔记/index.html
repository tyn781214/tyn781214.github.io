自学java的笔记

发表于 2019-11-28 更新于 2019-12-01

下文的对比是在将c语言与java部分作比较

java的编译器

我使用的java编译器是eclipse。除去JDK的下载以及Path配置比较麻烦之外，java可以在写代码的过程中检查语句错误。相比于c语言来说方便了很多。但相比于C语言来说java对于大小写相当敏感。

java语言的开始部分

c语言中一般由头文件放在最开始（如：#include<stdio.h>），而java则会有系统自动给出的开头。如果需要引用的文件则在最前面引用。（相当于c语言里的引用库函数吧）

java的语法与c语言的比较

1.关于bollean值

与C语言相似，java同样也有整型、浮点型、字符型，但条件判断句式后一定要跟boolean类型（如if(true)）。因为java里并不自动将0归为false、非0为true。所以这是一个与C语言不同的地方。java里的boolean只识别true和false。

2.输入与输出

C语言中的输入与输出是靠库函数<stiod.h>实现的，而在java中的输出是用System.out.print("想打印的字");实现的。二者的不同我暂时还不太清楚。
java的输入需要在public前加入import java.util.Scanner;这一串代码。
在使用时需要写入Scanner scan = new Scanner(System.in);后直接调用：String name = scan.nextLine();  int age = scan.nextInt();注：scan可以换成任意的符号。
